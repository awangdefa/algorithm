# 栈

###### 
栈也是一种数据呈线性排列的数据结构，不过在这种结构中，我们只能访问最新添加的数
据。栈就像是一摞书，拿到新书时我们会把它放在书堆的最上面，取书时也只能从最上面的新
书开始取。
